Raccoon Game

I participated with 4 of my friends in the 72 hours Ludum Game Jam whose theme this year is: start with nothing. Our game is a 2D puzzle game where user can drag and drop puzzle pieces into the game area to help the raccoon character achieve their goal.

I created multiple 2D environment prop pieces and small illustrations. With a total of 20 unique game level, we were placed 23rd in the overall ranking out of 2613 submission: in the top 1%! Currently we are in the process of revamping the UI and porting it to mobile!

Companion

Companion was created for GMTK Game Jam of 2019 whose theme was: "only one". Our game is a puzzle platformer where the player only has one box that they can befriend and use to clear multiple levels.

This was the first game jam I was in. With only 48 hours on the clock, I created all the UI, animation sprites, and environment pieces for the game. Our ranking wasn't as high as we had hoped for, but it was a fun experience that made me want to participate in more game jams! Currently in the process of redesigning the overall look!
